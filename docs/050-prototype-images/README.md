# Prototype Images

Bo anh SVG dung cho bao cao va trinh bay UI/UX cua Text-to-Comic App.

## Files

- `00-prototype-overview.svg`: Tong quan cac page chinh va luong nguoi dung.
- `01-dashboard.svg`: Dashboard danh sach project.
- `02-text-import.svg`: Man hinh nhap truyen chu va tao storyboard.
- `03-storyboard-editor.svg`: Storyboard editor, character casting, generate panel.
- `04-comic-editor.svg`: Comic editor voi speech bubble.
- `05-export-modal.svg`: Export PNG/PDF va trang thai thieu panel.

Tat ca file la SVG nen co the mo truc tiep bang browser, chen vao slide, hoac export sang PNG khi can.
